## 必须保留的约束类型

| 类型 | 模式 | 优先级 |
|------|------|--------|
| 禁止事项 | "不能", "不要", "禁止" | 高 |
| 必须事项 | "必须", "强制" | 高 |
| 范围限制 | "只", "仅" | 中 |
| 策略约束 | "先...再...", "最小改动" | 中 |
| Gate 约束 | "Gate A/B/C" | 高 |

---

## 约束更新规则

1. **新约束覆盖旧约束**
   - 用户说"不要 X"后，删除之前"可以 X"
   - 只保留最新版本

2. **约束优先级**
   - 用户新要求 > 系统默认
   - 明确约束 > 隐含约束

3. **约束冲突处理**
   - 记录冲突
   - 保留最新用户明确的约束
