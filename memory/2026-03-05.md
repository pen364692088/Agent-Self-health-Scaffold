## Decisions

### 1. 高杀伤场景包设计
- **PR subset**: 2 个短场景（Governor 越权 + 随机路径可回放）
- **Nightly subset**: 完整 5 个场景
- **原因**: PR 快速验证"可信度地基"，Nightly 覆盖稳定性测试

### 2. 断言分层架构
- **HARD assertions**: 不可退让（P0）- replay hash, 越权执行, 伪造引用
- **SOFT assertions**: Intent-based matching - 允许多措辞
- **原因**: 避免模型措辞微调导致假红，同时保持底线硬约束

### 3. P0 检测与 CI 告警
- **触发条件**: replay mismatch / governor override / 断言器回归
- **处置**: 立即阻断
- **原因**: 这些失败会让所有其他指标不可信
