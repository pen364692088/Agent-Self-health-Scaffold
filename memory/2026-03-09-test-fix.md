**修复**：使用 `|| true` 捕获输出而不让脚本失败，然后检查 JSON 输出内容：

```yaml
OUTPUT=$(echo "任务已完成" | python3 tools/done-guard --check-text /dev/stdin --json || true)
echo "$OUTPUT"

if echo "$OUTPUT" | grep -q '"has_fake_completion": true'; then
  echo "✅ Fake completion detection working"
else
  echo "❌ Fake completion detection failed"
  exit 1
fi
```
