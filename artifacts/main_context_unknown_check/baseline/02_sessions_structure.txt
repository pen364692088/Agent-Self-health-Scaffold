# Sessions.json Structure Analysis

## Key Finding

sessions.json 格式是扁平的 key-value 结构：
```
{
  "sessionKey1": { sessionData },
  "sessionKey2": { sessionData },
  ...
}
```

**不是** `{ "sessions": [...] }` 数组格式。
